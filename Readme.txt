○ゲーム説明
「操作説明」
移動->WASDキー
回避->移動+左Shift (必殺ゲージ使用)
攻撃->左クリック
武器切り替え->右クリック
必殺技発動->ホイールクリック(必殺ゲージが満タンの時)
ポーズ画面->戦闘中に左Ctrl
カメラの遠近調整->マウスホイール
視点切り替え->マウスの移動。

「MainMenuの操作説明」
モードの決定->左クリック

「必殺技」
発動中は無敵。
発動中に左クリック連打で、チャージ時間を短縮できる。

「ステータスバー」
上のゲージが、HP。
下のゲージが、必殺ゲージ。

「ミニマップ」
矢印のようなアイコンがPlayerで、向いている方向に先が向く。
赤の●が敵の位置。
緑の●がアイテムの位置。(常にあるとは限らない)
青の●がエネミーの位置。

「アイテム」
一定間隔で落ちてきて、HP回復と必殺ゲージ増加させる効果がある。
アイテムがある時は場所がミニマップに映されている。

「終了方法」
Escキーまたは、MainMenuでExitボタン左クリック。

「リザルトの評価基準」
Time -> クリア時間の速さ。(速いほど上)
Life -> 食らったダメージ量。(少ないほど上)
Item -> アイテムを取った量。(少ないほど上)
